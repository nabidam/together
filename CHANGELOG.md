# Changelog

All notable changes to Together are documented here.

## [0.2.1] - Unreleased

### Fixed

- Reject guest joins using a room link regenerated during the join request.
- Prevent stale playback starts from racing a confirmed media change.

## [0.2.0] - 2026-07-22

### Added

- Create rooms before choosing media.
- Let hosts choose or request a media change from inside a room.
- Require every current participant to confirm a media change before it applies.
- Show total media-library storage used in Admin.

### Changed

- Prefill upload titles from selected media filenames.

## [0.1.1]

- Latest released version before this changelog.
